Qt Canvas Module


The canvas module provides a highly optimized 2D graphic area called QCanvas. Upon the canvas, you place QCanvasItems and view the canvas with 1 or more QCanvasViews, which are QWidgets.

QCanvas features fast redrawing, animation, collision detection and more. Please see the documentation of the above classes for details.
