Chapter 6. The Menus And The Toolbars

The Window Menu

Close

Closes the currently active window.

Close All

Closes all windows, including the property editor.

Next

Activates the next window in the order in which the windows were created.

Previous

Activates the previous window in the order in which the windows were created.

Tile

Tiles all windows so that no two windows overlap.

Cascade

Cascades all windows so that only the title bar shows for all but the topmost window.

Property Editor

Raises or opens the property editor window.

Object Hierarchy

Raises or opens the object hierarchy window.

Form List

Raises or opens the form list window.
