Chapter 7. The Dialogs

The Pixmap Function Dialog

You will see this dialog if you specify in the Form Settings dialog that you want to use a pixmap-loading function and then select a pixmap (i.e., by inserting a pixmap label and choosing Choose Pixmap from the label's context menu). You can specify here the arguments that should be passed to your pixmap-loading function.
