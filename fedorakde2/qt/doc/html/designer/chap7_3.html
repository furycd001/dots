Chapter 7. The Dialogs

The Connection Viewer Dialog (Edit Connections)

Figure 7-3. The Connection Viewer dialog

This dialog is invoked by selecting Edit/Connections from the menu bar. It shows the connections between signals and slots that are in place for the currently active form. The central, unlabeled list box shows, for each connection, the outgoing GUI element, the signal name, the incoming GUI element, and the slot name.

Edit

Brings up the Connection Editor dialog (see the section called The Connection Editor Dialog (Edit Connections)), which lets you edit all connections between the outgoing and incoming GUI elements in the currently selected connection.

Disconnect

Removes the currently selected connection.
