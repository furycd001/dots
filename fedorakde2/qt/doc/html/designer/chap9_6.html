Chapter 9. The File Format Of .ui Files

The Description Of The Tab-Chain

The tab-chain of the form is defined in a <tabstops> element, which has any number of <tabstop> elements. Each of these specifies the name of the widget. There is one <tabstop> element for each widget in the form that can accept focus. The order of the <tabstop> specifies the order of the tab-chain.
