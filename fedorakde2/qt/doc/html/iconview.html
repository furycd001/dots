Qt IconView Module


The icon view module provides a powerful visualization widget dubbed QIconView. API and feature-wise it is similar to QListView and QListBox. It contains optinally labelled pixmap items that the user can select, drag around, rename, delete and more.

Please see the class documentation for details.
