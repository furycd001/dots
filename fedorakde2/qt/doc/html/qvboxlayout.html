Detailed Description

The QVBoxLayout class lines up widgets vertically.

This class provides an easier way to construct vertical box layout objects. See QBoxLayout for more details.

The simplest way to use this class is: 

     QBoxLayout * l = new QVBoxLayout( widget );
     l->addWidget( aWidget );
     l->addWidget( anotherWidget );

See also QHBoxLayout, QGridLayout and Layout overview documentation

Examples: layout/layout.cpp i18n/main.cpp scrollview/scrollview.cpp customlayout/main.cpp
